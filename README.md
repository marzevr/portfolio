# Personal Website
A personal web portfolio by Marios Evripidou containing multiple static HTML/CSS/JS microsites. Each section serves a unique purpose—from showcasing professional experience to presenting music, reflections, and utilities.

## Features

- Modular, static, fast-loading microsites
- Personal profile and project portfolio
- Music albums and streaming links
- Quotes and reflections gallery
- Tribute gallery for Zouzou the cat
- Minimalist New Tab homepage with search and bookmarks
- Poetry and personal reflections collection
- Mobile responsive design across all pages
- Fully self-contained — no backend or database required

## Requirements

- Modern web browser
- Optional: Web server or static hosting (e.g., Namecheap, GitHub Pages)
- Text editor for customization
- Internet connection (for streaming links, CDN fonts/icons)

## Dependencies

Most libraries are included locally or linked via CDN. Key dependencies include:

- [Google Fonts – Poppins & Inter](https://fonts.google.com/)
- [Feather Icons](https://feathericons.com/) – used in portfolio site
- Optional: Font Awesome (for potential icon use)

## Setup

```bash
git clone https://github.com/mariosevripidou/personal-website
cd personal-website
```

## How to Run

**Local Preview**  
Open any `.html` file in your browser, e.g.:

```bash
open index.html
```

**Web Deployment**  
Upload the directory tree to a static hosting service (e.g., Namecheap via SFTP).

## Files

- `index.html` – Portfolio and professional profile (Project Manager)
- `music/index.html` – Dream Pop / Ambient music albums and streaming links
- `newtab/index.html` – Custom homepage with search forms and direct bookmarks
- `quotes/index.html` – Personal quotes and reflections in a visual gallery
- `zouzou/index.html` – Memorial page and image archive of Zouzou the cat
- `poetry/index.html` – Poetry archive with anchor navigation
- `assets/`, `img/`, `css/`, `js/`, etc. – Local assets per microsite
